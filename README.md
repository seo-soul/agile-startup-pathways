# Agile Onboarding Pathways

A structured onboarding curriculum for developers joining an agile team. This repository walks you through the mindset, tools, practices, and assessments needed to contribute effectively from day one.

---

## How to Use This Repository

Work through the sections in order. Each file is designed to be read and discussed with your assigned mentor. Take notes, ask questions, and revisit sections as needed.

```
Week 1  →  Getting Started
Week 2  →  Agile Foundations
Week 3+  → Practices & Deep Dives
```

## Your First Week Checklist

- [ ] Complete the MBTI and PrinciplesYou assessments
- [ ] Share results with your mentor
- [ ] Set up your development environment (see WORKFLOWS.md)
- [ ] Read the Agile Manifesto and Principles
- [ ] Attend your first daily standup
- [ ] Meet your team and Scrum Master
- [ ] Review the Glossary and flag any unfamiliar terms
